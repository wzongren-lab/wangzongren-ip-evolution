## 一、底层认知（道）

### 1. 成分 vs 环节

- 钩子、骨架、情绪刺点是结构的三大**成分**，不是三个**环节**
- 成分 = 可叠加、可出现在任何位置
- 环节 = 有先后顺序（错误认知，会导致学死知识）

### 2. 动力结构 vs 模板结构

- 模板结构：表面形式（第一段讲书、第二段讲内容...）——**无效**
- 动力结构：找到背后的原因，用什么动力抓住你往下看——**有效**
- 能抓人的不是形式排列，是动力

### 3. 人群适配

- 钩子骨架情绪刺点因人群而异，要为目标用户定制
- 结构影响流量：什么开头吸引什么人，什么骨架留住什么人
- 整体思考：内容+人设+钩子+人群必须匹配

### 4. 钩子可叠加

- 一种语言可同时触发两种动力
- 画面+语言可同时作用
- 不是3秒内只能说一句话所以只能用一种钩子——动力可以叠加

## 二、13种钩子（术）

| 序号 | 钩子名称 | 本质 | 核心手法 |
|------|----------|------|----------|
| 1 | 不稳定状态登场 | 带着故事登场 | 表情高情绪、魔术师口吻、故事关键环节、镜头高情绪、动作高情绪、惊叹 |
| 2 | 完形心理（并置） | 让观众想象两者关系 | 视觉并置、语言并置，越不可能结合越想看 |
| 3 | 与我相关 | 让目标用户觉得有关 | 点名人群、直接对话、用户场景（非产品介绍）、痛点式包装、美好正向结果 |
| 4 | 否定性力量 | 利用损失厌恶和恐惧 | 重大盲区、危害纠错、常识颠覆、撕开认知缺口 |
| 5 | 社会身份敏感 | 人对阶层排序的敏感 | 引起愤怒骄傲、加强优越感、自我贬低（双刃剑慎用） |
| 6 | 动物性本能 | 唤醒本我 | 食欲情欲、听觉刺激（沉浸式/气泡音）、视觉刺激（高饱和色/闪光）、触觉唤醒（视觉引发生理反应）、危险疼痛预警 |
| 7 | 围观与热闹 | 人本能想看热闹 | 人多、跟班、热闹场面；可叠加动物性欲望（一群帅哥美女） |
| 8 | 窥探 | 满足偷窥欲 | 自我暴露、隐私创作、行业内幕 |
| 9 | 反常（反差） | 熟悉场景中的陌生感 | 不可能任务、特殊生活要强调差异点、冲突不可调和、数字对比强调（需对比双方同时呈现） |
| 10 | 特殊消息（资讯） | 满足好奇心 | 热议话题、新概念新名词、传奇故事、平凡人英雄之旅——与反常的区别：好奇 vs 惊奇 |
| 11 | 强律动（节奏） | 人对节奏的向往 | 卡点、爆款BGM、强节奏动作；骨架中也极重要 |
| 12 | 人间美好 | 人对美好的向往 | 美景、美好结局、奋斗成功——谁都能用，必须会用 |
| 13 | 新奇有趣 | 满足娱乐需求 | 脑洞挑战、趣味游戏、荒诞怪诞——需天赋，可遇不可求 |

### 钩子关键认知

- 13个不是穷举，是经验汇聚，可能有第14个
- 第1个（不稳定状态）最重要最有效，希望每条视频都有
- 第12个（人间美好）如果前面做不到，至少做到这个
- 第13个（新奇有趣）放最后，因为需天赋
- 钩子只解决前2-5秒的注意力抓取，不决定整体流量

## 三、情绪刺点（刺激反应的巴普洛夫食物）

**核心原则**：到处预埋，多多益善。大多数人刷抖音是冷静理智疏离的，必须刺激到情绪阈值才会互动。

### 点赞刺点

| 手法 | 说明 |
|------|------|
| 动机美好 | 照顾小狗、帮助人、发心善良 |
| 过人之处 | 让人佩服的成功经历 |
| 氛围感美 | 画面美、风景美、构图美 |
| 突如其来高潮 | 刺激反转、情感突变、形式变调 |
| 伴随笑声 | 嘴角上扬时容易点赞 |
| 结尾高光 | 冲破成功、大团圆、震撼金句 |

### 评论刺点

| 手法 | 说明 |
|------|------|
| 提问引导 | "你们还有什么别的补充的" |
| 埋梗 | "懂得都懂" |
| 站队 | 为谁说话、二元对立 |
| 露出破绽 | 给专业的人去指责 |
| 话说一半 | 保留信息保持未知，提高完播率+懂王帮你回复 |
| 可复制句子 | "留下你的时间11:11" |

### 收藏刺点

| 手法 | 说明 |
|------|------|
| 实用技巧 | 以后用得上 |
| 值得学习的风格 | 行事风格 |
| 过载信息量 | 看不懂先收藏以后再看 |

### 转发刺点

| 手法 | 说明 |
|------|------|
| 社交货币 | 谈资 |
| 教育目的 | "你看吧这个人说的多好" |
| 互相关切 | 炫耀、激励 |
| 诱因设计 | "快分享给你的谁" |

### 关注刺点

| 手法 | 说明 |
|------|------|
| 工具价值 | 有价值栏目、实用陪伴 |
| 精神价值 | 活成了别人想活的样子 |
| 身份背书 | 强调过往经验履历 |
| 关注后续 | 且听下回分解 |
| 小众差异化 | 越小众越值得被关注 |
| 人格魅力 | 声音好听、亲切、善良 |
| 能人所不能 | 做到行业最被诟病事情的反面 |

**能人所不能的关键**：问自己——你这个行业最讨人厌的点是什么？反向做到并强调出来。

## 四、6种骨架（如何留住观众，解决完播率）

**核心认知**：99%没有数据的原因不是因为情绪刺点，而是骨架不够好。骨架是结构中最难最重要的部分。

### 1. 兴趣点推进

- **定义**：时刻感觉到新鲜和有意思，来不及无聊，持续的信息喂养
- **核心形式**：信息枚举、清单式
- **为什么有效**：颗粒度清晰（AB分明）、数字提醒有安全感（像导航）、容错率更高、N+1情绪刺点
- **进阶**：生活切片聚合——每个切片都是不稳定状态登场（钩子），聚合后成为兴趣点推进（骨架）
- **信息密度公式**：信息密度 = 信息总量 ÷ 时间。过快看不懂=0，过啰嗦=0，过简单犯困，过难也犯困

### 2. 戏剧性推进

- **定义**：时刻感觉到紧张悬念，对结局有期待
- **四种悬念方式**：
  1. 希区柯克式：明知道结局，想看怎么走到结局（路的拐角/桌底炸弹）
  2. 暗示期待结局：构建期待后不断设置障碍（不行、还不行、还是不行，直到最后释放）
  3. 情节重新理解：不断引出新信息让过去重新理解（难度极高）
  4. 关心人物移情：好人受难、深度同情（宠物/小孩/老人/弱势群体适用）
- **关键场面要彻底释放**。宁可过火不可不过。"从物理学上不需要踢七脚，但创作中的真实不是现实的真实"

### 3. 共鸣性推进

- **定义**：时刻感觉到情感力量，想到自己，有所感悟
- **核心路径**：好奇 → 感兴趣 → 看进去 → 想到自己 → 被打动 → 共鸣
- **关键**：从特殊到一般。先讲小众独特生活（凑热闹），再讲到所有人都有的事件（共鸣）。反过来不行——上来就讲普遍的事，会觉得不新鲜
- **两大要素**：
  - 具象细节（基石）：行动、事件、资讯，所见即所得
  - 抽象总结（画龙点睛）：告诉观众该怎么理解
- **比例原则**：具象为主，抽象为辅。可以没有总结，但不能没有具体事情。三步一喘息
- **核心认知**：观众不喜欢听道理，喜欢在故事中找到自己的答案。同一套材料可以讲出完全不同的故事

### 4. 逻辑性推进

- **定义**：时刻有新认知，在逻辑上反转，认同或不认同
- **两种方向**：
  1. 二元对立引发站队：先扮演反方（先抑后扬），认同的点赞、不认同的评论、摇摆的被说服
  2. 单方面碾压：疯狂轰炸认知，不解释只告知
- **常用技巧**：不断的自我否定（本以为→但是、有人说→但是）
- **注意**：这只是表面话术，没有动力再多的自我否定都没用

### 5. 节奏型推进

- **定义**：时刻感觉到律动，有洗脑感
- **入手最容易，上限也最高**
- **表现形式**：卡点、排比（当你→你会）、押韵、长短句（短句短句短句长句→呼吸感）、内在的气（吊着一口气奔涌向前）
- **音乐处理**：提前找好，写文案时听着音乐改，句子长短跟着音乐走；音乐本身也需剪辑，转折处卡上内容变化

### 6. 销售型推进

- **定义**：成交的心路历程，让观众完成购买决策
- **适用范围最小**，因为本质是要买东西
- **六个步骤**（可任意组合，一般从1或2开始）：
  1. 激发兴趣：用钩子或画面吸引多看两眼
  2. 创造需求：正向展示美好场景 / 反向展示痛点
  3. 卖点罗列：结合热点、构建场景
  4. 赢得信任：权威证明、顾客证言、畅销证明
  5. 化解忧虑：算账、化整为零、限时优惠、售后保障
  6. 下单指令：直接指挥行动
- **决策成本与链路长度**：越高决策成本链路越长，越低决策成本链路越短

## 五、骨架三大意识

### 1. 观众意识

- 知道观众什么时候看不下去
- 适时总结、聊天、打趣、变道，疏密有致，张弛有度
- 牵着观众的手过河，体谅之心

### 2. 颗粒度意识

- 信息密度：AB分明，不要黏糊糊一团
- 不要总想着过渡自然，学会大的跳跃
- 转折否定就是最好的跳跃
- 不要过度解释，不要分支太多
- 一句话告诉观众要干嘛，头也不回地往这个方向讲

### 3. 势能意识

- 所有时间的艺术都是积累和释放的艺术
- 任何不相关的信息都会削弱势能
- 过程比结果重要，不要蔓延编织枝叶
- 情未到浓时不会牵手
- **核心原则**：不要告诉我结果，告诉我思考的过程。不要告诉我你的感受，带我重新感受一遍

## 六、创作流程

1. **随便做**：先有初稿，哪怕稀巴烂的
2. **选择骨架**：用一种主要骨架重新梳理
3. **尝试叠加**：叠加其他动力，量力而行
4. **设定时限**：改3天就发，不能提前不能拖迟

核心原则：学会多少用多少，不要等全部学会。四个钩子够玩半年。

---

## 七、道法术的关系

- **道**：规律。说不清，但存在
- **法**：根据道定下来的大方针。不变，必须有流量结构、必须有深层动力
- **术**：灵活变化的招数。会变，钩子可能过时，但背后的法不会
- **结论**：术是经验而非真理。通过术掌握背后的法，是永远的能力。平台会变，法不变

## 八、关键概念锚点（课程反复强调）

1. 短视频的基本功，所有人都需要
2. 摆地摊不意味着可以不认真做视频
3. 不是因为有钩子才去做内容，而是已有想法再去加钩子完善
4. 拿来主义，能用多少用多少
5. 用户的场景（非产品介绍），所有东西要考虑用户的场景
6. 每一个观众都是利己的，无利不起早
7. 信息密度 = 信息总量 ÷ 时间
8. 过程比结果重要
9. 关键场面要充分释放，宁可过火不可不过
10. 钩子只解决前2-5秒，不决定整体流量
