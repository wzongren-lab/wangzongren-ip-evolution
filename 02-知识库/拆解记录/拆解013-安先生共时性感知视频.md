# 拆解013-安先生共时性感知视频

> 拆解日期：2026-04-25
> 视频主题：历时性与共时性的短视频创作方法论
> 核心概念：共时性并置 vs 历时性修补

---

## 文案全文

```
为什么说左边的视频比右边的视频设计更为精心

为什么这条视频要在工地里拍

为什么这条视频要在饭店里拍

为什么这条视频要在教室里拍

为什么同样一个团队的人都要用相似的方式拍摄

为什么这个博主在演讲台上也要讲相似的话

回答这些问题之前

我想先区分人类获取信息的两种感知模型

历时性

共时性

我想你知道

我们做短视频一定要快

观众如果感觉无聊

就会划走

所以你希望在横向的时间轴上去塞进无穷多的信息量

47 秒 13 个建议

67 秒 10 个时间

如果你稍微高级一点

你可能会知道信息量的种类绝对不是单一的

信息量的把玩还可以是多材质的

一条视频中穿插图片、表情包、手机拍摄画面、相机拍摄画面、DV 画面

但是

如果这就是你理解的信息的全部传递方式

我想你就必须把这条视频看完了

在历时性的感知里

我们触碰一个又一个的信息

我们在信息的缝隙中填补我们的思绪

这种修补所需要的想象力常常伴随着恍惚

伴随着不同的人在不同的状态下不同程度的游离

当我们用历时性感知对事物的缝隙进行填补时

无可避免的会遗留出新的缝隙

因此，才可以激发的情感强度必然是间接的，也必然不够强烈，也必然需要观众高度的注意力

在这个时间和注意力都被疯狂争夺的时代

我们的心灵越来越繁忙

修补的能力也越发稀缺

我们在生活中

除非有重大事件

我们很难高度的集中

我们的注意力

而打开抖音这些短视频平台

观看短视频

并非我们需要高度集中注意力的事物

加之我们前面所讲

视频在被任意的传播

随时会闯入不合时宜的人的生活里

这种主动的修补就更加难以发生

所以我给我的学生说

比起在横向上下功夫

不如在纵向上给予更多的可能性

在视觉的深处

在一瞬间

将事物并置

激发起一种共时性的感知

在电影史上

《公民凯恩》的著名镜头里

导演并非历时性的在时间轴上先出现苏珊，再出现安眠药的画面，最后再单独出现凯恩闯进房间里发现一切的画面

而是在刹那间

让安眠药、苏珊和凯恩共时性的出现在一个画面里

在短视频里

同样在奏效

妻子和丈夫的并置

母亲和孩子的并置

电影和乐器的并置

然而对共时性的理解

不应该只在内部的视觉层面

更要想办法构建时空的完整性

所以黄老师比起桃子老师更擅长与画外空间进行互动

他必须暗示台下有人

这条视频也必须和台下有互动

最早的影像

全部来自于想象

音轨上的补充只是提示主题

辅助精神修补的音乐

宛如我们在看书的时候所听的背景音乐一样

而背景的环境声则是一个巨大的飞跃

这种飞跃不是技术性上的，而是美学上的、时空观上的更新

当背景环境声出现的时候

则是在告诉我们

我们得到的是一个连续性的空间和时间

窗外的车水马龙一刻都未曾停歇

历时性的观看需要人们集中注意力，对精神进行主动的修补

而共时性的并置则直接的闯入人们的心灵

人们对近在眼前的事物是无法视而不见的

看见了就是看见了

共时性的并置强势的让人们无法拒绝

而非可怜巴巴的渴望观众能主动参与、渴望观众能幻想

在短视频里

这种姿态往往成为了必需

也成为了一种必然的策略

历时性的观看容易让观众走马观花、抽身事外

而共时性的感知则让人望眼欲穿

把目光扎进时间的纵深
```

## 八、对律师IP的迁移价值

### 8.1 可直接使用的手法

| 手法                               | 迁移方式                      | 律师内容示例                                                                                                                   |
| ---------------------------------- | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| **六连问风暴开场**           | 用6个"为什么"围绕一个法律问题 | "为什么这个当事人赢了？为什么那个当事人输了？为什么证据一样结果不同？为什么律师说法不一样？为什么法官这样判？为什么上诉没用？" |
| **概念对举**                 | 用一对概念构成论证骨架        | "程序正义vs实体正义""形式合规vs实质合规""合同约定vs法律强制"                                                                   |
| **学术概念下沉**             | 引用法学理论但用案例解释      | "正当防卫"来自刑法理论，用具体案例解释                                                                                         |
| **点名对比触发站队**         | 点名具体案例或判决进行对比    | "A案判赔100万，B案同样情况判赔10万，差异在哪？"                                                                                |
| **"我给当事人说"降低说教感** | 把法律建议变成经验分享        | "我经常给当事人说，比起盯着合同条款，不如盯着合同履行过程"                                                                     |
| **美学意义层**               | 从法律技术上升到社会意义      | "合规不是技术问题，是企业价值观问题"                                                                                           |

### 8.2 律师IP的具体场景

**场景1：建设工程纠纷**

- 六连问开场："为什么这个工期索赔被支持？为什么那个工期索赔被驳回？为什么同样的停工函效力不同？为什么鉴定结果差这么多？为什么律师说法不一样？为什么二审经常改判？"
- 概念对举："工期顺延vs工期延误""签证效力vs会议纪要效力"

**场景2：税务合规**

- 六连问开场："为什么这个企业被稽查？为什么那个企业安然无恙？为什么同样的业务税负差这么多？为什么律师和会计师说法不一样？为什么税务筹划变成偷税？为什么合规成本这么高？"
- 概念对举："税务筹划vs偷税""形式合规vs实质合规"

**场景3：民事执行**

- 六连问开场："为什么有的案子能执行到位？为什么有的案子终本了？为什么同样的财产线索处置结果不同？为什么律师和法官说法不一样？为什么执行异议这么多？为什么追加被执行人这么难？"
- 概念对举："执行到位率vs终本率""形式财产vs实际财产"

### 8.3 律师IP的语气建议

本条视频的语气是"教师式+美学家式"：

- 教师式：教方法论（怎么做共时性并置）
- 美学家式：从技术上升到美学意义

**律师IP可采用的语气组合**：

- 教师+法律专家：教法律方法+解释法律意义
- 避免：过于说教，用"我给当事人说"代替"你应该"

## 十、总结

本条视频是安先生方法论的高阶应用：

1. **开场**：六连问风暴，制造认知缺口暴雨
2. **骨架**：逻辑性推进为主，叠加兴趣点/戏剧性/共鸣性
3. **势能**：钻井式势能，从技术层到美学层到哲学层
4. **钩子**：全程持续，六连问开场+概念命名+点名对比+结尾升华
5. **语气**：教师式+美学家式，专业深度+美学高度

**对律师IP的核心启发**：

- 六连问风暴是最适合律师IP的开场方式之一——法律问题本身就有很多"为什么"
- 概念对举可以让法律论证更有张力和记忆点
- "我给当事人说"可以把法律建议从说教变成经验分享
