# 拆解001：安先生"爆款短视频完整模型"

> 拆解时间：2026-04-24
> 文案来源：安先生本人的教学视频
> 文案性质：方法论讲解视频，视频本身即为方法论的最佳示范

---

## 文案全文

每一个人都可以在自媒体时代成名一次，只需要他了解爆款短视频的完整模型。这就是爆款短视频的完整模型。

现在从最初的起点开始。首先短视频是一则飘荡在互联网的游离的信息，和专门去电影院看电影或者在聊天中给朋友看所拍的视频不一样，短视频缺乏前后的语境，随机的闯入猝不及防的生活，于是导致了三个任意：任意对象、任意场景、任意状态。这三个任意就是短视频语法的基石。

每一条短视频都必须在开端携带人群印记，在公域流量池筛选自己的精准用户，就如同在茫茫的沙子中寻找铁钉，人群印记就是寻找铁钉的吸铁石。它可以是视觉性的，想想人群会看到什么而挪不开眼睛；它也可以是语言性的，想想人群会听到什么字眼而兴奋。对的人群印记会让对的人留下，留下才有可能点赞评论，而这些互动数据会实时的影响着视频的推送走向，所以稍不留神推送模型就会跑歪。

然而只有精准的用户观看只是爆款短视频的必要条件，而非充分必要条件，所以必须建立流量的漏斗模型，必须从精准用户扩散到边缘群体，必须有更多的围观人群来推热视频。这种扩散往往体现在选题上的扩散，所以比起谈论如何解决腹直肌分离问题，不如谈论产后如何恢复好身材；比起谈论为什么正山小种是最好的红茶，不如扩散为最好的 10 种茶叶排名。

但是只要瞄准了精准用户做内容就一定会想看吗？想想最开始所讲的三个任意中的最后一个：任意状态。目标用户并非时时刻刻都是观看的好状态，所以常常指导学生要用人的本能进行注意力的抓取。例如人们都喜欢看人多热闹的地方，这就是流量密码。再比如比起得到什么，人们更害怕失去什么，这就是流量密码：损失厌恶。再比如人们都渴望在事物和事物之间发现联系，这就是流量密码：完型心理。再比如人们都喜欢卷入正在发生的事情，这就是短视频的 "切片性"。

以上所讲的都只是为了解决爆款短视频的前两秒，然而下一步才是更为关键的，让观众持续兴奋到最后一秒。这条视频总时长 5 分钟，在 2 分钟和 4 分钟的准点分秒不差的提醒观众别走。这条视频 13 个心得，46 秒讲完。这条视频 20 秒介绍 5 个卖点，40 秒展示 6 个场景。

好莱坞电影的节拍表以 10 分钟为一个单位，短视频的节拍表以 3 秒钟为一个单位。请记住短视频的短从来都不是总时长的短，而是创作者的时间粒度变短，信息的间距变短。不要问要拍 40 秒还是 60 秒，而要问能拍多少个 3 秒。短视频的时间呈现原子化，而非黏糊糊的一团。

短视频是一场时间的赛跑，每一个时间的原子都必须奔涌向前，永不停歇。短视频如果有时态，那一定是正在进行时。短视频的创作者比谁都有压力，必须时时刻刻构建一种来不及完成的当下，每一个当下都必须快速成为过去，以便让新的当下到来，时间的张力必须保持高张状态。

一个段落在被彻底完结之前应该被提前剪辑掉，意思在被解释清楚之前应该有新的信息进入，悬念不应该彻底落地，就应该被提前转移走，或者由新的悬念来接棒。一条视频必须表现为不断涌现的持续的时间，这就是时间性压力。

就像这条我经常讲的视频，最后的照片从专业角度来讲并非惊艳之作，然而却获得百万点赞。随着过程形成一种奔涌向前的势能，情绪得以被不断累积，时间性压力会转化成情绪的动力。当人们进入高情绪的状态时，严苛冷漠的标准便会放低，行动力会被进一步激活，点赞评论转发分享关注便更有可能。

短视频从来都不是一种单向的言说行为，它发生在人们的任意空间中，偶遇你的任意状态，所以不存在完美的交流。就像正在看这条视频的你，可以完全不明白它在讲什么，但是它是一个心智的材料，它是一个刺激物，它刺激你看完并付诸点赞评论等互动行为，进而推动这条视频登上热门。

所以常常给学生说这条视频的 13 个心得是刺激观众 13 次点赞的机会，是 13 次评论的机会，也是 13 次关注的机会。如果把点赞评论的赌注压在最后，那一定是冒险的，因为观众未必能看到最后，而刺激物的预埋则是一种提高胜算的做法。

人们在哪里观看短视频并不重要，人们所处的空间任意化、虚无化，所处的空间彻底退场，让观看过程中的时间体验成为唯一，时间内在于短视频之中，成为构建短视频秩序的语法。每一个观众的不同时间感受都会影响着自己何时离开，是否看完，是否做出点赞评论等互动行为，进而改变着短视频的生命进程。

电影导演塔可夫斯基曾经认为电影是一门雕刻时光的艺术，然而短视频更是如此。鲜明地成为了时间的团块，电影的时间由导演雕刻，而短视频的时间由创作者和观众共同雕刻。因此与其问什么是短视频，不如问什么是短视频的时间语法。与其问如何创作短视频，不如先知悉如何使用时间这门语言来和观众进行心灵的交流。

所以曾经讲过的重复、差异、停顿、缝隙、否定以及切片都是时间的语法。

## 钩子分析（前2秒）

开篇一句话叠加了**至少4种钩子**：

> "每一个人都可以在自媒体时代成名一次，只需要他了解爆款短视频的完整模型。"

| 钩子类型 | 如何生效 |
|----------|----------|
| 与我相关 | "每一个人"——把所有刷到的人都圈进来 |
| 人间美好 | "成名一次"——人对成功的向往 |
| 否定性力量/认知缺口 | "只需要了解…"——暗示你还不知道这个模型，你有一个重大盲区 |
| 完形心理 | "完整模型"——给了你一个形状，但里面是空的，你必须看下去填充 |

一句话四重动力同时作用，是课程强调的"钩子可以叠加"的顶级示范。

## 全文钩子分布

课程强调：钩子是成分不是环节，可以出现在任何位置。这条视频全程每隔几秒就有钩子。

| 位置 | 内容 | 钩子类型 |
|------|------|----------|
| 开头 | "每一个人都可以在自媒体时代成名一次" | 与我相关（每一个人）+人间美好（成名） |
| 开头 | "只需要他了解爆款短视频的完整模型" | 否定性力量/认知缺口 + 完形心理（完整模型） |
| 定义段 | "三个任意：任意对象、任意场景、任意状态" | 反常（概念命名）+ 节奏型推进（排比短句） |
| 人群印记段 | "在茫茫的沙子中寻找铁钉" | 完形心理（并置：沙子和铁钉） |
| 人群印记段 | "稍不留神推送模型就会跑歪" | 否定性力量（恐惧损失） |
| 转折段 | "**然而**只有精准的用户观看只是...必要条件，**而非**充分必要条件" | 逻辑性推进-自我否定：制造新的不确定 |
| 扩散段 | "比起谈论如何解决腹直肌分离问题，**不如**谈论产后如何恢复好身材" | 与我相关（痛点具体化）+ 反常（对比） |
| 本能抓取段 | "人们都**喜欢**看人多热闹的地方...人们**更害怕**失去什么...人们都**渴望**发现联系...人们都**喜欢**卷入正在发生的事情" | 围观与热闹 + 否定性力量（损失厌恶）+ 完形心理 + 不稳定状态（切片性）——**四种本能钩子排比叠加** |
| 节奏段 | "这条视频**13个心得**，**46秒**讲完...**20秒**介绍**5个卖点**，**40秒**展示**6个场景**" | 节奏型推进（数字节拍）+ 兴趣点推进（信息密度展示） |
| 时间性压力段 | "每一个时间的原子都必须**奔涌向前**，**永不停歇**" | 节奏型推进（短句加速）+ 不稳定状态（持续运动） |
| 时间性压力段 | "一个段落在被彻底完结之前应该被**提前剪辑掉**" | 反常（违反直觉的认知） |
| 案例段 | "就像**正在看这条视频的你**，可以完全不明白它在讲什么" | 与我相关（直接对话）+ 破第四面墙 |
| 案例段 | "这条视频的13个心得是刺激观众**13次**点赞的机会" | 数字锚定 + 情绪刺点（重复提醒） |
| 结尾升华段 | "与其问什么是短视频，**不如**问什么是短视频的时间语法" | 逻辑性推进（自我否定升级） |
| 结尾收束 | "重复、差异、停顿、缝隙、否定以及切片都是时间的语法" | 话说一半（6个概念不展开）+ 节奏型推进（概念排比） |

**关键发现**：这条视频不是开头抓一下就完，而是**全程每隔几秒就有钩子**。尤其是"四种本能钩子排比"那一段，连续4个短句，每句都是一种钩子类型——这不是简单的重复，是钩子的**机枪扫射**，确保无论你是什么类型的人，总有一款击中你。

## 骨架分析

### 主骨架：逻辑性推进（单方面碾压路线）

论证链路层层递进，每个环节都是对前一个的否定/追问：

```
短视频的本质（三个任意）
    ↓
第一个问题：怎么找到对的人？→ 人群印记
    ↓
第二个问题：对的人就够了吗？→ 不够，要扩散（流量漏斗）
    ↓
第三个问题：瞄准了就一定看吗？→ 不一定，状态不对（任意状态）→ 本能抓取
    ↓
第四个问题：前2秒抓住了然后呢？→ 持续兴奋到最后一秒
    ↓
核心揭示：时间性压力 → 情绪累积 → 行动激活
    ↓
升华：短视频=雕刻时间 → 塔可夫斯基
```

每个环节的"不够"都是课程中逻辑性推进"不断的自我否定"技法的运用。

### 叠加骨架1：节奏型推进

- 短句密集区："任意对象，任意场景，任意状态。这三个任意就是短视频语法的基石。"
- 短→长加速区："每一个当下都必须快速成为过去，以便让新的当下到来"
- 排比："再比如…再比如…再比如…"（损失厌恶、完形心理、切片性三个流量密码排比）
- 数字节奏："13个心得46秒""5个卖点40秒""6个场景"——数字本身就是节拍器

### 叠加骨架2：戏剧性推进

"然而只有精准的用户观看只是爆款短视频的必要条件，而非充分必要条件"——希区柯克式悬念：你以为知道结局（找到精准用户=成功），但告诉你不够，必须看怎么走到结局。

后半段"时间性压力"概念是整条视频的关键场面释放，前面所有铺垫都在为此积累势能。

### 叠加骨架3：共鸣性推进

从抽象到具体，再从具体到抽象：

- 抽象："游离的信息""三个任意"
- 具体："腹直肌分离→产后恢复""正山小种→10种茶叶排名""13个心得46秒"
- 抽象升华："雕刻时光""时间的语法"

这是共鸣性推进"从特殊到一般"的变体——先建立认知框架，用具体案例落地，再用哲学高度升华。

## 骨架三大意识体现

### 观众意识
- "在2分钟和4分钟的准点分秒不差的提醒观众别走"——承认观众会走，主动留人
- "稍不留神推送模型就会跑歪"——不是教训你，是替你焦虑

### 颗粒度意识
- "13个心得46秒""5个卖点40秒""6个场景"——数字精确，颗粒度清晰
- "不要问拍40秒还是60秒，而要问能拍多少个3秒"——AB分明，不黏糊

### 势能意识
- 整条视频没有一句废话，所有信息都在为"时间性压力"这个核心概念积累势能
- "当人们进入高情绪的状态时，严苛冷漠的标准便会放低"——这句话本身就在对观众执行它所描述的效果

## 课程未明确覆盖的实战手法（补充点）

### 1. 自我指涉/元叙事
视频在讲解爆款模型的同时，自身就是爆款模型的实施。观众不是被说服的，是被体验的。课程没有专门讲这种"内容即演示"的手法。

### 2. 概念命名即钩子
"三个任意""时间性压力""时间的语法"——把复杂认知包装成可传播的概念标签。这超出课程13个钩子的范畴，更接近选题层面的能力。

### 3. 哲学升华作为关注刺点
用塔可夫斯基结尾，把"术"提升到"道"，让觉得"学到了技巧"的观众突然觉得"这个人有思想"。课程讲了关注刺点7种类型，但这种从工具价值跨越到精神价值的瞬间转换，课程没有专门论述。

### 4. 论证节奏（逻辑的韵律感）
课程讲节奏型推进提到卡点、排比、长短句，但这条视频展示了一种更高级的节奏——论证节奏。逻辑递进本身就是节拍：前提→否定→追问→揭示→升华。这种"逻辑的韵律感"课程没有展开。

## 对律师IP的迁移价值

- **不断追问结构**：建工纠纷/税务合规内容，可用类似结构——"合同签了就安全？不够。安全了就能拿到钱？不够。拿到钱了就不用交税？不够。"
- **时间性压力**可直接用到执行领域——"执行有时效，每过一天都在流失"，这本身就是势能积累
- **概念命名**：在专业领域造词——"执行三任意""税务合规的漏斗模型"，用概念标签做钩子
