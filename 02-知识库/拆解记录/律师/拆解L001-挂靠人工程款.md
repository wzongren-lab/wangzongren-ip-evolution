## 一、钩子落地对照

### 开篇叠加

第1句："挂靠人能要求被挂靠人支付工程款吗？"

叠加了3种钩子：
- **与我相关**：点名人群"挂靠人"，直接触达目标用户
- **否定性力量**：暗示"可能不能"，制造认知缺口，触发恐惧损失
- **特殊消息**：具体的法律问题，有资讯价值

第2-3段叠加：
- **与我相关**：继续点名"实际施工人"
- **窥探**：展示真实咨询案例，行业内幕感
- **否定性力量**："败诉的风险是很大的"，恐惧损失
- **完形心理**：挂靠关系vs转包关系，制造认知对比

"以我12年工程案件办理经验"叠加：
- **特殊消息**：权威背书
- **社会身份敏感**：12年经验，专业身份标签

### 全程钩子分布

| 位置 | 钩子类型 | 具体表现 |
|------|----------|----------|
| 开篇提问 | 与我相关+否定性力量+特殊消息 | 点名人群+制造认知缺口 |
| 案例引入 | 窥探+与我相关 | 真实咨询案例+点名实际施工人 |
| 风险警示 | 否定性力量 | "败诉的风险是很大的" |
| 概念对比 | 完形心理 | 挟靠关系vs转包关系 |
| 经验背书 | 特殊消息+社会身份敏感 | 12年经验 |
| 实战案例 | 窥探+人间美好 | 亲办案件+胜诉结局 |

**验证"钩子是成分不是环节"**：全程持续布钩，不只在开头。

## 二、骨架落地对照

### 主骨架：逻辑性推进

推进链条：
1. 问题提出 → 认知缺口
2. 风险警示 → 恐惧势能积累
3. 知识输出（三个关键点） → 部分释放
4. 实战案例 → 再次积累（如何赢的？）
5. 诉讼策略 → 释放
6. 当庭技法 → 高潮释放（原告自认）
7. 法条背书 → 权威释放
8. 结局 → 完全释放

逻辑性推进特征：不断自我追问（能要求吗→为什么不能→如何应对→如何赢），二元对立（挂靠vs转包、支付义务vs转付义务）。

### 辅骨架：兴趣点推进

清单式输出：
- 三个关键点（第一、第二、第三）
- 证据准备清单（投标介入证据、投标保证金来源、聊天记录）
- 法条引用清单（建工解释一第43条全文、证据规定第5条、民诉法解释第92条）

**验证"骨架可叠加"**：逻辑性推进（主）+ 兴趣点推进（辅）。

## 三、势能意识落地对照

### 核心概念

挂靠关系vs转包关系的法律后果区别——决定实际施工人能否胜诉的关键。

### 势能积累释放路径

| 段落 | 势能状态 | 分析 |
|------|----------|------|
| 问题提出 | 积累 | 制造认知缺口 |
| 风险警示 | 积累 | "败诉风险很大"制造恐惧 |
| 三个关键点 | 部分释放 | 知识输出，但保留悬念（如何应用） |
| 亲办案例引入 | 积累 | 再次制造悬念（如何赢的？） |
| 诉讼策略 | 部分释放 | 策略揭示，但保留技法悬念 |
| 如何证明 | 积累 | 新悬念（证据准备+当庭技巧） |
| 当庭技巧 | 高潮释放 | 原告律师自认，戏剧性转折 |
| 法条背书 | 权威释放 | 法律依据支撑 |
| 结局 | 完全释放 | 胜诉，判决采纳 |

### 是否有削弱势能的废话

"这类的咨询我解答过很多，我亲办的这类案件也很多"有轻微重复，但服务于建立权威身份，可接受。

整体势能波浪结构：积累→部分释放→积累→高潮释放→完全释放。

## 五、首尾呼应落地对照

### 开头概念

问题："挂靠人能要求被挂靠人支付工程款吗？"

### 结尾回归

答案（通过案例结局呈现）：被挂靠人对挂靠人不承担支付责任，只在收款范围内承担转付义务。

**闭环特征**：问题→答案，但不是简单回答"不能"，而是通过完整案例证明"为什么不能+如何应对"，增加了情感重量（胜诉案例印证）。

## 律师IP可迁移模式

### 开篇模式：提问+咨询案例式开场

**具体写法**：
- 第1句：直接抛出法律问题（"XX能要求YY做ZZ吗？"）
- 第2句：用真实咨询案例承接（"这是最近处理的一个咨询问题"）
- 第3句：描述案例细节（人物+行为+困境）
- 第4句：点出法律风险（"如果搞不清楚……败诉风险很大")

**可迁移到**：建工纠纷（挂靠、转包、违法分包）、税务合规（稽查风险）、民事执行（执行异议）

**迁移示例**：
- 税务合规："企业能拒绝税务机关的发票调取要求吗？这是最近处理的一个咨询……"
- 民事执行："债权人能直接执行债务人的配偶财产吗？这是最近处理的一个咨询……"

### 骨架推进模式：问题→风险→知识点→案例→技法→结局

**具体写法**：
1. 问题提出（制造认知缺口）
2. 风险警示（恐惧势能积累："败诉风险很大"/"罚款风险很大"/"执行失败风险很大"）
3. 知识点输出（清单式：三个关键点/三个要点）
4. 亲办案例承接（再次积累悬念："我刚办结的一个XX案件"）
5. 技法揭示（高潮释放：诉讼策略+证据准备+当庭技巧）
6. 法条背书（权威释放：引用具体法条全文）
7. 结局（完全释放：胜诉/成功）

**可迁移到**：所有执业方向的干货输出类内容

**迁移要点**：
- 第3步知识点要用清单式（第一、第二、第三），形成兴趣点推进
- 第5步技法要有戏剧性（当庭技巧、突袭策略、关键证据），制造高潮
- 第6步法条必须引用全文，增强收藏价值

### 情绪刺点模式：法条全文+证据清单式

**具体写法**：
- 收藏刺点：在知识点段落引用法条全文（如建工解释一第43条完整内容），在技法段落列举证据清单（投标介入证据、投标保证金来源、聊天记录）
- 点赞刺点：在结局段展示胜诉成果，在技法段展示过人之处（提醒书记员记关键话）
- 评论刺点：开篇提问引导
- 关注刺点：结尾段展示执业年限+专业定位（"执业XX年，专办XX案件")

**可迁移到**：所有干货类内容，尤其是需要建立专业权威的IP

**迁移要点**：
- 法条引用必须写明条文全称及完整内容（符合律师工作规则）
- 证据清单要有具体例子，不能只说"准备证据"
- 结尾身份标签要精确（执业年限+专业方向）
