## 课程核心理论

（以下内容来自课程，拆解时对照验证）

### 一、钩子理论

**核心认知**：钩子是成分不是环节

- 可以出现在任何位置，不是开头抓一下就完
- 全程持续布钩，钩子密度决定留存
- 钩子可叠加：一种语言可同时触发多种动力

**课程提及的钩子类型**：
- 与我相关（点名人群）
- 人间美好（向往）
- 否定性力量（恐惧损失/认知缺口）
- 反常
- 完形心理
- 围观与热闹
- 不稳定状态
- 社会身份敏感
- 特殊消息
- 窥探
- 资讯
- 权威参照
- 机制揭秘

### 二、骨架理论

**6种骨架类型**：
1. 逻辑性推进：不断自我否定/追问，推向下一步
2. 兴趣点推进：信息枚举、清单式
3. 共鸣性推进：从特殊到一般，让观众照见自己
4. 节奏型推进：时长短句排比押韵，有洗脑感
5. 戏剧性推进：通过冲突推动

**骨架三大意识**：
1. 势能意识：所有时间的艺术都是积累和释放的艺术，任何不相关的信息都会削弱势能
2. 颗粒度意识：AB分明，不要黏糊糊一团，不要总想着过渡自然，学会大的跳跃
3. 观众意识：知道观众什么时候看不下去，适时总结、聊天、打趣、变道，疏密有致

**骨架可叠加/切换**：不只用一种骨架，多种骨架无缝切换

### 三、情绪刺点理论

**核心原则**：到处预埋，多多益善

- 大多数人刷短视频是冷静理智疏离的，必须刺激到情绪阈值才会互动
- 五类刺点：点赞、评论、收藏、转发、关注
- 话说一半：不把话说完，观众自己填，触发互动

### 四、首尾闭环

- 开头的概念在结尾回归
- 回归时增加情感重量，不是简单重复

## 形式原则库

（从拆解中提炼，跨领域可迁移）

### 开篇模式

| 模式名称 | 结构特征 | 适用场景 |
|---------|---------|---------|
| 全称+条件式 | 全称词开篇 + 条件式暗示缺口 → 与我相关+人间美好+否定性力量+完形心理叠加 | 方法论教学、概念定义 |
| 条件+定义式 | "当我开始……，我……，今天我明白了，这就叫XX" | 情感共鸣类、价值观输出类、金句类 |
| 动作打断+当下体验点名 | 不稳定状态登场（正在发生的动作）+ 与我相关（点名用户当下正在经历的体验） | 任何需要让观众"进入"的场景 |
| 哲理名句开篇 | 名人名句/哲理性句子开篇 → 制造认知缺口 | 知识输出、审美教育、价值观传递 |
| 否定性概念定义 | 否定常识（"不是XX"）+ 抛出新概念（"而是YY"）+ 制造认知冲突+好奇心 | 概念类、认知升级类、打破固有认知的主题 |
| 概念定义式开篇 | 第1句抛出新概念名词 + 第2句连接目标人群利益 + "如果...就..."假设句式制造悬念 | 方法论教学、概念定义、需要建立专业权威的IP |
| 热点话题+身份刺痛+否定性力量 | 抛出热点话题 + 用隐喻形象刺痛观众身份 + 隐含否定性力量（你可能理解错了） | 打破观众对热门话题的肤浅认知、知识类观点类内容 |
| 时代背景+哲学提问 | 点名时代特征（人人都在XX）+ 抛出本质追问（到底意味着什么）+ 与我相关+特殊消息+完形心理叠加 | 价值观输出类、行业反思类、概念定义类 |
| 问题叠加式 | 连续多个"为什么"形成问题群 + 暗示认知缺口 + 抛出新概念 | 方法论教学、概念定义、需要制造紧迫感的知识输出 |
| 提问+自我暴露+技法预告 | 抛出热议问题 + 自我暴露身份/背景 + 技法预告 | 方法论教学类内容、需要建立专业权威的IP、目标用户想学具体技法 |
| 热点话题+否定性力量+反常数字 | 抛出热点话题 + 精确数字制造反差（如"七分钟290次运镜"）+ 否定性力量（你可能理解错了）| 打破观众对热门话题的肤浅认知、价值观传递类、热点解读类 |
| 个人经历自嘲式 | 第1句：我+过去梦想（崇高职业）+ 第2句：但+现实打断（自嘲式理由）+ 叠加钩子：与我相关+完形心理+人间美好+反常（自嘲）| 情感共鸣类、价值观输出类、治愈类、需要让观众快速代入的内容 |
| 否定性自我暴露式开篇 | 以"我有一个偏见/一个判断"开头 + 用否定性词汇自我定义 + 抛出专业判断标准 | 需要建立专业身份、抛出新概念、制造认知冲突时 |
| 视觉冲击+否定性定义 | 视觉元素开场（颜色/画面）+ 否定常识（"这和……截然不同"）+ 概念定义（"这是……"）+ 价值承诺（"流量密码"） | 方法论教学、概念定义、需要制造视觉冲击的知识类内容 |
| 时代特征+隐含否定式 | 第1句时代特征判断（"这个世界正在越来越XX"）+ 第2句具象案例枚举 + 第3句转移到人际领域（隐含身份敏感）+ 隐含否定（你追求的可能意味着失去）| 价值观输出类、社会批判类、审美教育类 |
| 群体状态追问式 | 第1句点名群体"我们" + 状态追问"都是怎样" + 第2句抛出核心概念（如"丧失了表达欲"）+ 叠加钩子：与我相关+否定性力量+完形心理+特殊消息 | 情感共鸣类、状态描述类、戳痛点类内容 |
| 反常识权威否定式 | 第1句引用"玩笑话/俗语/现象"作为切入点 + 第2句反转指出权威人物犯错 + 第3句建立悬念"为什么权威错了？" + 叠加钩子：特殊消息+反常+否定性力量+身份敏感 | 需要建立新认知、颠覆常识的内容、哲学思辨类 |
| 行动宣言式 | 第1句："我决定+今晚+做XX的人" + 用"我决定"制造主观意志 + 用"今晚"制造时间紧迫感 + 用"XX的人"制造身份标签 | 情感共鸣类、价值观输出类、需要快速建立人设/气质的内容 |
| 精确数字+体验追问式 | 第1句：精确数字开篇（如"七年""五点"）+ 第1句末：体验式追问（"是一种怎样的体验？"）+ 叠加钩子：反常+特殊消息+与我相关+完形心理 | 个人经历分享类、生活方式类、习惯养成类内容 |
| 痛点场景叠加式 | 第1句：用"当你……"直接代入用户场景 + 第2句：描述该场景下的具体行为细节 + 第3句：隐含否定性力量（"仿佛……才得体"暗示被规训）| 情感共鸣类、痛点戳穿类、状态描述类内容 |

### 骨架推进模式

| 模式名称 | 结构特征 | 适用场景 |
|---------|---------|---------|
| 不断否定式 | A→不够→B→不够→核心概念→升华；每步用否定推动；中间插入数字节拍 | 论证类、概念定义类 |
| 共鸣叠节奏式 | 共鸣性推进为主骨架（让每个人代入）+ 节奏型推进为辅助骨架（排比句式洗脑感） | 情感共鸣类、价值观输出类、治愈类 |
| 枚举积累→情感释放 | 多个技术维度枚举积累势能→转向人生意义释放情感势能 | 知识类内容，需要在结尾升华为情感共鸣 |
| 双峰共鸣结构 | 开篇提出概念→故事/细节积累势能→第一次情感释放（金句）→再次积累势能→第二次情感释放（核心金句）→实用价值释放 | 情感共鸣类、需要金句传播、概念→实践的内容 |
| 概念递进式推进 | 概念提出→案例堆叠（多维度验证）→常识颠覆→权威背书→概念深化→艺术升华→收束回归；中段有明确承上启下信号 | 方法论教学、传递复杂概念、需要建立认知高度的IP |
| 否定链推进+场景释放交替 | 抛出伪认知A→否定→抛出伪认知B→否定→抛出伪认知C→否定；每次否定后用用户场景释放势能；最终抵达核心认知 | 观众有多个常见误区需纠正、观点类认知升级类内容 |
| 否定→定义→升华 | 否定常识定义 → 提出精神定义 → 具象案例深化 → 新维度展开 → 批判对比 → 连接价值 → 自我认知 → 行动召唤 → 终极定义 | 概念定义类、价值观传递类 |
| 枚举积累→哲理释放→行动清单 | 大量案例枚举积累势能→哲理金句释放情感势能→具体行动清单落地 | 知识输出类、方法论教学类、需要同时满足收藏价值和行动指引的内容 |
| 概念批判式推进 | 抛出问题群→引入概念区分→批判现状→提出方案→权威背书→延伸应用→姿态升华 | 传递新概念、纠正固有认知、建立专业高度的知识类内容 |
| 技法枚举+案例示范+情感升华 | 技法枚举（多个手法逐一展开）+ 每个技法配案例示范 + 从技法→情感价值升华 + 结尾提问收束 | 知识输出类、方法论教学类、需要收藏价值+情感共鸣、适合专业IP建立权威感 |
| 热门案例枚举→本质追问→普通人枚举→核心定义 | 热门案例细节枚举→本质追问（为什么）→其他爆款案例枚举→普通人生活枚举→核心概念定义→行动召唤 | 价值观输出类、情感共鸣类、需要让观众"照见自己"的内容 |
| 个人→群体→反转升华式 | 个人经历枚举（2-3个，形成节奏）+ 归类群体（"我们这样的人"）+ 反转（"但这从不妨碍我们……"）+ 升华收束 | 情感共鸣类、需要从个体经验扩展到群体共鸣的内容 |
| 逻辑+共鸣双轨推进 | 前半段逻辑性推进（概念→问题→理论）+ 后半段共鸣性推进（个人故事→普遍痛点→升华） | 需要兼顾认知输出和情感共鸣时 |
| 概念→案例→理论→共鸣四段式 | 概念定义 → 艺术案例解释 → 学员数据验证 → 心理学理论深化 → 人生场景升华 → 情感收束 | 传递复杂概念、需要从专业层穿透到情感层的深度内容 |
| 精确日期节拍式 | 连续案例用精确日期串联（X月X日→X月X日→X月X日），形成时间节拍感 | 方法论教学、需要用数据验证概念的内容 |
| 批判追问→反向论证→权威背书→信念升华式 | 现象描述→因果追问→批判反思→反向论证（案例证明相反观点）→权威背书→信念表达→升华收束 | 价值观输出类、审美教育类、打破固有认知的观点类内容 |
| 场景枚举+节奏节拍+共鸣收束式 | 多个具体场景枚举（本篇6个），每个场景"想A→但B"结构 + 场景间形成重复节拍 + 中段插入权威参照（古诗词）提升势能 + 归类群体"我们总是……"完成从特殊到一般的跃迁 + 核心定义释放→行动召唤收尾 | 情感共鸣类、痛点描述类、需要让观众"照见自己"的内容 |
| 叙事积累→共鸣释放→反高潮收束 | 第1阶段：具体行为叙事，细节枚举积累势能 + 第2阶段：转向用户痛点场景，共鸣性推进，情感金句释放势能 + 第3阶段：突然收束，反高潮（不是升华，而是"就这样"），留白 | 情感共鸣类、治愈类、需要营造"余韵"而非"金句轰炸"的内容 |
| 四段式哲学落地推进 | 理论层（引用权威/哲学观点）→ 案例层（名人案例佐证）→ 个人层（自身经历证明）→ 落地层（回归用户场景） | 需要将抽象概念落地到实用场景的内容、哲学思辨类、方法论教学类 |
| 个人经历→群体共鸣→权威升华式 | 第一阶段：个人经历叙述（我七年前做了什么）+ 第二阶段：归类群体（"XX的朋友都知道"）+ 第三阶段：认知升华（早起的价值不在本身）+ 第四阶段：权威背书释放（名人金句） | 生活方式分享类、习惯养成类、价值观传递类内容 |
| 压抑积累→宣言释放→余韵收束 | 第一阶段：多个"当你……你……"句式，形成压抑感的节奏重复（势能积累）+ 第二阶段：用"可是/但我真的想……"打破节拍，开始释放势能 + 第三阶段：排比句式"哪怕……哪怕……"形成宣言式高潮（势能充分释放）+ 第四阶段：古诗词/名言收束，不再积累新势能，余韵收尾 | 情感共鸣类、价值观输出类、需要从痛点走向理想的内容 |
| 中西对比枚举→场景召唤→文化升华式 | 第1阶段：多个中西对比案例枚举（名人+古文），每例形成"西方A→中国B"结构，积累认知冲突势能 + 第2阶段：转向用户场景"如果你需要……"，给出行动方案，释放实用价值 + 第3阶段：古诗词收束 + 文化概念升华（文化自信→自我重塑） | 文化自信类、价值观传递类、打破固有认知的观点类内容 |

### 情绪刺点模式

| 模式名称 | 结构特征 | 适用场景 |
|---------|---------|---------|
| 预告+话说一半 | 中段明确预告互动机会；结尾概念不展开留缺口 | 方法论教学、干货输出 |
| 金句密度式 | 每段结尾都有定义式金句，全篇可摘录收藏 | 金句类、治愈类、价值观输出类 |
| 留白式话说一半 | 呈现"是什么"，不解释"怎么做"，留给观众自己填 | 情感共鸣类、引发思考类 |
| 全程信息密度+结尾升华 | 收藏刺点全程分布（信息密度大）+ 点赞/转发刺点结尾释放（人间美好升华） | 知识类内容，需要同时满足收藏价值和情感共鸣 |
| 美学向往刺点 | 全程用美好/崇高词汇（超验、崇高、永恒、上帝视角）触发点赞和收藏 | 审美教育类、价值观传递类 |
| 金句+书签式 | 金句埋在势能释放点（点赞转发）+ 实用清单放在结尾（收藏）+ 全程气质统一建立关注动机 | 知识分享类、价值观输出类、需要收藏价值的内容 |
| 分层预埋模式 | 点赞刺点在收尾段（金句/氛围感美）+ 收藏刺点在中段理论段（方法论）+ 转发刺点在洞察段（社交货币）+ 关注刺点全文渗透（专业度）+ 评论刺点在常识颠覆段（站队引导） | 知识型/教学型内容、需要高完播率的深度内容、专业IP人设建设 |
| 痛点场景+古文金句+话说一半 | 用痛点场景触发点赞/评论 + 用古文金句触发收藏 + 用话说一半触发评论（不说破，让观众自己代入） | 知识类观点类内容、目标用户有一定文化水平、需要社交货币属性 |
| 信息过载+名家清单式 | 全程高密度信息喂养（收藏刺点）+ 名家名单形成收藏清单（收藏价值最大化） | 知识输出类、资源推荐类、方法论教学类 |
| 金句插入+话说一半 | 哲理金句作为势能释放点（点赞/转发刺点）+ 呈现不解释（评论刺点） | 审美教育类、价值观传递类 |
| 定义密度式 | 每段都有可独立摘录的定义式金句，全篇可拆解收藏 | 概念类、价值观类、金句类 |
| 三隐喻叠加式 | 结尾用三个隐喻叠加（是A，是B，是C），形成情感高潮 | 需要强升华效果的结尾 |
| 概念密度+美学释放式 | 全程高密度概念喂养（收藏刺点）+ 结尾美学画面感释放（点赞/转发刺点）| 审美教育类、方法论教学类、需要收藏价值的内容 |
| 技法清单+意象清单+提问收束 | 技法清单触发收藏（方法论）+ 意象清单触发点赞/转发（情感共鸣）+ 提问收束触发评论（观众参与） | 方法论教学类、需要同时满足收藏价值和情感共鸣、目标用户有学习动机+情感需求 |
| 身份敏感词+金句收尾式 | 中段：身份敏感词（如"社畜"）触发评论/共鸣 + 结尾：金句式收尾（如"仰望星空"）触发点赞/转发 + 全篇：诗意气质统一建立关注动机 | 情感共鸣类、治愈类、价值观输出类 |
| 本质追问+氛围感美+价值观金句 | 本质追问触发评论思考 + 细节枚举营造氛围感美（点赞）+ 结尾价值观金句触发转发/关注 | 情感共鸣类、价值观传递类、治愈类 |
| 理论收藏+金句转发+痛点评论 | 理论框架段落（方法论）+ 美学金句（社交货币）+ 痛点共鸣段落 | 知识型/美学型内容需要多维度互动时 |
| 心理学内幕+人生金句 | 中段用心理学理论满足收藏需求 + 后段用人生金句触发转发/点赞 + 结尾用邀请式收束触发评论 | 知识类情感类融合内容、需要建立专业权威+情感共鸣的IP |
| 话说一半+邀请代入 | 呈现人生场景但不解释（"我们曾经渴望……然而最后却发现"）+ 结尾邀请观众参与（"如果我有幸与你携手闯过"） | 情感共鸣类、需要观众代入自己的故事 |
| 艺术家清单+名家引言+金句密度式 | 艺术家名单枚举（收藏刺点——信息密度）+ 名家引言嵌入（转发刺点——社交货币）+ 金句分布在势能释放点（点赞刺点——氛围感美）| 审美教育类、价值观传递类、需要收藏价值+社交货币属性的知识类内容 |
| 话说一半+氛围画面+金句定义式 | 中段用莫名其妙的内容触发好奇和评论 + 多个氛围感画面触发点赞 + 定义式金句触发收藏和转发 + 收尾陪伴式建议触发关注 | 情感共鸣类、治愈类、需要观众代入和互动的内容 |
| 氛围感美+话说一半 | 氛围感美（点赞）：用美好画面（送花）+ 情感金句（"晚上的世界是你一个人的"）触发 + 社交货币（转发）：用用户痛点场景（白天的烦躁）触发 + 话说一半（评论）：结尾戛然而止，不解释"抱着自己睡觉意味着什么"，留白 | 情感共鸣类、治愈类、诗意气质类 |
| 金句铺垫+提问收尾式 | 中段密集铺设点赞刺点（动机美好+高光金句）+ 提供可收藏的认知框架（如角色三分法）+ 结尾用提问引导评论 | 需要同时触发点赞、收藏、评论的内容、哲学思辨类 |
| 金句定义+群体点名式 | 点赞刺点：结尾权威金句（动机美好+品格向往）+ 评论刺点：中段"经常XX的朋友都知道"点名群体触发站队 + 收藏刺点：认知升华段落（方法论可迁移）+ 转发刺点：金句本身有社交货币属性 + 关注刺点：能人所不能的意志力展示 | 生活方式类、价值观输出类、需要建立专业/自律人设的IP |
| 痛点场景+古诗词金句+话说一半式 | 前半段：痛点场景触发评论/点赞（观众代入"我就是这样"）+ 中段：价值观宣言触发转发（社交货币）+ 收尾：古诗词引用触发收藏 + 话说一半触发评论（不解释"意味着什么"）| 情感共鸣类、治愈类、诗意气质类内容 |
| 名人清单+古文金句+文化升华式 | 收藏刺点：全程高密度名人名言+古文金句（信息过载）+ 点赞刺点：励志金句+文化认同感动+美好向往 + 转发刺点：认知反转+身份认同 + 评论刺点：开头提问+反转处话说一半 + 关注刺点：全文知识密度+文化底蕴建立专业权威 | 知识输出类、价值观传递类、需要收藏价值+社交货币的内容 |
