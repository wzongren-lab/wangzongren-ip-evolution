## 课程核心理论

（以下内容来自课程，拆解时对照验证）

### 一、钩子理论

**核心认知**：钩子是成分不是环节

- 可以出现在任何位置，不是开头抓一下就完
- 全程持续布钩，钩子密度决定留存
- 钩子可叠加：一种语言可同时触发多种动力

**课程提及的钩子类型**：
- 与我相关（点名人群）
- 人间美好（向往）
- 否定性力量（恐惧损失/认知缺口）
- 反常
- 完形心理
- 围观与热闹
- 不稳定状态
- 社会身份敏感
- 特殊消息
- 窥探
- 资讯
- 权威参照
- 机制揭秘

### 二、骨架理论

**6种骨架类型**：
1. 逻辑性推进：不断自我否定/追问，推向下一步
2. 兴趣点推进：信息枚举、清单式
3. 共鸣性推进：从特殊到一般，让观众照见自己
4. 节奏型推进：时长短句排比押韵，有洗脑感
5. 戏剧性推进：通过冲突推动

**骨架三大意识**：
1. 势能意识：所有时间的艺术都是积累和释放的艺术，任何不相关的信息都会削弱势能
2. 颗粒度意识：AB分明，不要黏糊糊一团，不要总想着过渡自然，学会大的跳跃
3. 观众意识：知道观众什么时候看不下去，适时总结、聊天、打趣、变道，疏密有致

**骨架可叠加/切换**：不只用一种骨架，多种骨架无缝切换

### 三、情绪刺点理论

**核心原则**：到处预埋，多多益善

- 大多数人刷短视频是冷静理智疏离的，必须刺激到情绪阈值才会互动
- 五类刺点：点赞、评论、收藏、转发、关注
- 话说一半：不把话说完，观众自己填，触发互动

### 四、首尾闭环

- 开头的概念在结尾回归
- 回归时增加情感重量，不是简单重复

## 形式原则库

（从拆解中提炼，跨领域可迁移）

### 开篇模式

| 模式名称 | 结构特征 | 适用场景 |
|---------|---------|---------|
| 全称+条件式 | 全称词开篇 + 条件式暗示缺口 → 与我相关+人间美好+否定性力量+完形心理叠加 | 方法论教学、概念定义 |
| 条件+定义式 | "当我开始……，我……，今天我明白了，这就叫XX" | 情感共鸣类、价值观输出类、金句类 |
| 动作打断+当下体验点名 | 不稳定状态登场（正在发生的动作）+ 与我相关（点名用户当下正在经历的体验） | 任何需要让观众"进入"的场景 |
| 哲理名句开篇 | 名人名句/哲理性句子开篇 → 制造认知缺口 | 知识输出、审美教育、价值观传递 |
| 否定性概念定义 | 否定常识（"不是XX"）+ 抛出新概念（"而是YY"）+ 制造认知冲突+好奇心 | 概念类、认知升级类、打破固有认知的主题 |
| 概念定义式开篇 | 第1句抛出新概念名词 + 第2句连接目标人群利益 + "如果...就..."假设句式制造悬念 | 方法论教学、概念定义、需要建立专业权威的IP |
| 热点话题+身份刺痛+否定性力量 | 抛出热点话题 + 用隐喻形象刺痛观众身份 + 隐含否定性力量（你可能理解错了） | 打破观众对热门话题的肤浅认知、知识类观点类内容 |
| 时代背景+哲学提问 | 点名时代特征（人人都在XX）+ 抛出本质追问（到底意味着什么）+ 与我相关+特殊消息+完形心理叠加 | 价值观输出类、行业反思类、概念定义类 |
| 问题叠加式 | 连续多个"为什么"形成问题群 + 暗示认知缺口 + 抛出新概念 | 方法论教学、概念定义、需要制造紧迫感的知识输出 |
| 提问+自我暴露+技法预告 | 抛出热议问题 + 自我暴露身份/背景 + 技法预告 | 方法论教学类内容、需要建立专业权威的IP、目标用户想学具体技法 |

### 骨架推进模式

| 模式名称 | 结构特征 | 适用场景 |
|---------|---------|---------|
| 不断否定式 | A→不够→B→不够→核心概念→升华；每步用否定推动；中间插入数字节拍 | 论证类、概念定义类 |
| 共鸣叠节奏式 | 共鸣性推进为主骨架（让每个人代入）+ 节奏型推进为辅助骨架（排比句式洗脑感） | 情感共鸣类、价值观输出类、治愈类 |
| 枚举积累→情感释放 | 多个技术维度枚举积累势能→转向人生意义释放情感势能 | 知识类内容，需要在结尾升华为情感共鸣 |
| 双峰共鸣结构 | 开篇提出概念→故事/细节积累势能→第一次情感释放（金句）→再次积累势能→第二次情感释放（核心金句）→实用价值释放 | 情感共鸣类、需要金句传播、概念→实践的内容 |
| 概念递进式推进 | 概念提出→案例堆叠（多维度验证）→常识颠覆→权威背书→概念深化→艺术升华→收束回归；中段有明确承上启下信号 | 方法论教学、传递复杂概念、需要建立认知高度的IP |
| 否定链推进+场景释放交替 | 抛出伪认知A→否定→抛出伪认知B→否定→抛出伪认知C→否定；每次否定后用用户场景释放势能；最终抵达核心认知 | 观众有多个常见误区需纠正、观点类认知升级类内容 |
| 否定→定义→升华 | 否定常识定义 → 提出精神定义 → 具象案例深化 → 新维度展开 → 批判对比 → 连接价值 → 自我认知 → 行动召唤 → 终极定义 | 概念定义类、价值观传递类 |
| 枚举积累→哲理释放→行动清单 | 大量案例枚举积累势能→哲理金句释放情感势能→具体行动清单落地 | 知识输出类、方法论教学类、需要同时满足收藏价值和行动指引的内容 |
| 概念批判式推进 | 抛出问题群→引入概念区分→批判现状→提出方案→权威背书→延伸应用→姿态升华 | 传递新概念、纠正固有认知、建立专业高度的知识类内容 |
| 技法枚举+案例示范+情感升华 | 技法枚举（多个手法逐一展开）+ 每个技法配案例示范 + 从技法→情感价值升华 + 结尾提问收束 | 知识输出类、方法论教学类、需要收藏价值+情感共鸣、适合专业IP建立权威感 |

### 情绪刺点模式

| 模式名称 | 结构特征 | 适用场景 |
|---------|---------|---------|
| 预告+话说一半 | 中段明确预告互动机会；结尾概念不展开留缺口 | 方法论教学、干货输出 |
| 金句密度式 | 每段结尾都有定义式金句，全篇可摘录收藏 | 金句类、治愈类、价值观输出类 |
| 留白式话说一半 | 呈现"是什么"，不解释"怎么做"，留给观众自己填 | 情感共鸣类、引发思考类 |
| 全程信息密度+结尾升华 | 收藏刺点全程分布（信息密度大）+ 点赞/转发刺点结尾释放（人间美好升华） | 知识类内容，需要同时满足收藏价值和情感共鸣 |
| 美学向往刺点 | 全程用美好/崇高词汇（超验、崇高、永恒、上帝视角）触发点赞和收藏 | 审美教育类、价值观传递类 |
| 金句+书签式 | 金句埋在势能释放点（点赞转发）+ 实用清单放在结尾（收藏）+ 全程气质统一建立关注动机 | 知识分享类、价值观输出类、需要收藏价值的内容 |
| 分层预埋模式 | 点赞刺点在收尾段（金句/氛围感美）+ 收藏刺点在中段理论段（方法论）+ 转发刺点在洞察段（社交货币）+ 关注刺点全文渗透（专业度）+ 评论刺点在常识颠覆段（站队引导） | 知识型/教学型内容、需要高完播率的深度内容、专业IP人设建设 |
| 痛点场景+古文金句+话说一半 | 用痛点场景触发点赞/评论 + 用古文金句触发收藏 + 用话说一半触发评论（不说破，让观众自己代入） | 知识类观点类内容、目标用户有一定文化水平、需要社交货币属性 |
| 信息过载+名家清单式 | 全程高密度信息喂养（收藏刺点）+ 名家名单形成收藏清单（收藏价值最大化） | 知识输出类、资源推荐类、方法论教学类 |
| 金句插入+话说一半 | 哲理金句作为势能释放点（点赞/转发刺点）+ 呈现不解释（评论刺点） | 审美教育类、价值观传递类 |
| 定义密度式 | 每段都有可独立摘录的定义式金句，全篇可拆解收藏 | 概念类、价值观类、金句类 |
| 三隐喻叠加式 | 结尾用三个隐喻叠加（是A，是B，是C），形成情感高潮 | 需要强升华效果的结尾 |
| 概念密度+美学释放式 | 全程高密度概念喂养（收藏刺点）+ 结尾美学画面感释放（点赞/转发刺点）| 审美教育类、方法论教学类、需要收藏价值的内容 |
| 技法清单+意象清单+提问收束 | 技法清单触发收藏（方法论）+ 意象清单触发点赞/转发（情感共鸣）+ 提问收束触发评论（观众参与） | 方法论教学类、需要同时满足收藏价值和情感共鸣、目标用户有学习动机+情感需求 |
